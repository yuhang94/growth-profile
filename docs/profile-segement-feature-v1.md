## 三、本期实现范围（v1）

本期聚焦 **行为事件圈选条件**，为现有分群条件树增加行为筛选叶子节点：

1. **做过（DID）**：用户在指定时间范围内做过某事件，可选属性过滤和次数条件
2. **未做过（DID_NOT）**：用户在指定时间范围内未做过某事件
3. **事件属性过滤**：对事件的 JSON properties 字段进行条件过滤
4. **时间范围**：支持绝对时间和相对时间（last_7d、last_30d 等）
5. **次数条件**：做过事件的次数满足 >=、>、=、< 等比较

### 条件 JSON 示例

```json
{
  "operator": "AND",
  "children": [
    { "tagKey": "vip_level", "compareOp": "GE", "values": ["3"] },
    {
      "behaviorOp": "DID",
      "eventName": "purchase",
      "propertyFilters": [
        { "propertyKey": "amount", "compareOp": "GT", "values": ["100"] }
      ],
      "timeRangeStart": "last_30d",
      "timeRangeEnd": "now",
      "countOp": "GE",
      "countValue": 3
    },
    {
      "behaviorOp": "DID_NOT",
      "eventName": "refund",
      "timeRangeStart": "last_30d",
      "timeRangeEnd": "now"
    }
  ]
}
```
